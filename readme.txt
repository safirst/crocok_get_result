===========================
crocok_get_result v.0.0.3b
===========================

Программа для получения результатов тестирования пользователей на КРОК-Экзамен.

Результат сохраняется в трех файлах (по числу попыток) в текущем каталоге.

По всем вопросам обращаться к asenkovskiy@croc.ru
